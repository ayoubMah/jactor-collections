#  MyCollection Framework — Roadmap v0.0.3

## Release Title: "Linked Foundations"

**Goal:** Implement and test a fully functional `MyLinkedList<E>` as the second `MyList` implementation.  
**Branch:** `dev/v0.0.3`  
**Duration:** ~1 week (depending on pace)

---

##  High-Level Objective

Build a doubly linked list that extends `MyAbstractList<E>` and demonstrates pointer-based storage, full CRUD operations, and iterator traversal — ready to serve as the backbone for future `Deque` and `Queue` implementations.

---

##  Development Tasks

### 1️⃣ Class Setup

- Create `MyLinkedList<E>` under `src/main/java/ayoub/list/`
- Extend `MyAbstractList<E>`
- Declare internal fields:
    - `private Node<E> head;`
    - `private Node<E> tail;`
    - `private int size;`
- Add inner static class `Node<E>`:
```java
private static class Node<E> {
    E data;
    Node<E> next;
    Node<E> prev;
}
```

### 3️⃣ Support Methods (internal/private helpers)

- [x] `private Node<E> nodeAt(int index)` — optimized traversal (if `index < size/2` → start from `head`, else from `tail`)
- [x] `private void linkBefore(E e, Node<E> succ)` — insert before a node
- [ ] `private void unlink(Node<E> n)` — detach node cleanly
- [ ] `clear()` — reset all references to `null`
